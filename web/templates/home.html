Quick Start

Install

curl -L https://github.com/username/jprq/releases/latest/download/jprq-linux-amd64 -o jprq
chmod +x jprq
sudo mv jprq /usr/local/bin/

Usage

# HTTP tunnel
jprq http 3000

# TCP tunnel
jprq tcp 22
